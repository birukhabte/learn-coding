# 📘 CodeLearn (Beta)

## 🚀 Overview

This repository contains the source code for a **CodeLEarn** currently in **beta**. The goal of this project is to bring multiple academic resources—such as tutorials, quizzes & Notes at One Place For Students**.

This project is  under development. I work On it When I Get Free From My Works 

---

## 🎯 Purpose of This Repository

The main purpose of this repository is to:

* Build a **centralized learning platform** for students
* Provide **structured tutorials** with easy navigation
* Enable **practice-based learning** through quizzes
* Organize **notes & Quizes** in one place


This repository can be updated or may be not
---

## ✨ Current Features (Beta)

* 🔐 **User Authentication**

  * Signup and login functionality

* 📚 **Tutorials Section**

  * Structured, read-only tutorials
  * Sidebar-based navigation

* 📝 **Quiz Section**

  * Topic-wise quizzes (basic implementation)
  * Instant score feedback

* 👤 **Profile Section**

  * Displays basic user information
  * Limited editing support (beta)
